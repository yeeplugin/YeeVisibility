Yeekit Visibility Documentation

Yeekit Visibility is a comprehensive WooCommerce plugin giving you granular control over product visibility and access permissions. Restrict products by user role, location, login status, and more.

Key Features

  • Flexible Targeting →

    Apply rules to Products, Categories, Tags, Brands, or Attributes.

  • Advanced Conditions →

    Filter by User Role, Geo Location, Cart Total, Purchase History, etc.

  • Search Protection →

    Automatically exclude hidden products from search and catalog queries.

Installation

  1. Download the `yeevisibility.zip` file.
  2. Go to WordPress Admin > Plugins > Add New.
  3. Upload the plugin ZIP file and click Install Now.
  4. Activate the plugin.

Configuration

Settings are located at: WooCommerce > Settings > Product Visibility.

The interface has two main tabs:

  • Rules: Manage specific visibility rules.
  • Settings: Configure global behavior like Search Hiding and Redirect URLs.

Creating Rules

Go to the Rules tab and click Add Rule.

Rule Components

  • Filter Type: What are you targeting? (e.g., specific Category).
  • Filter Value: Which items? (e.g., "Wholesale" category).
  • Action:
    • Hide Completely: Remove from shop and search.
    • Restrict Access: Show in catalog but block single product page.
  • Conditions: Who does this apply to? (See below).

Condition Types

Combine conditions to create powerful logic.

  • User Role

    Target specific roles (e.g. Wholesale, VIP).

  • Geo Location

    Restrict by Country or State.

  • Cart Logic

    Based on Cart Total or Items in Cart.

  • User History

    Based on products previously purchased.

Global Settings

Control plugin-wide behavior.

  • Hide from Search: Remove hidden products from search results.
  • Hide from Catalog: Remove form Shop/Category pages.
  • Global Redirect URL: Default destination for restricted users.
  • Allow Search Bots: Let Google/Bing index hidden content (use with caution).

FAQ

Q: Why are products still visible in search?
A: Check Global Settings > "Hide from Search". Also clear your site cache.

Q: Can I schedule visibility?
A: Yes, use the "Date & Time" condition in your rule.
